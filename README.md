# Kulineri API

- API Endpoint : https://lazy-shorts-fish.cyclic.app/
- API Documentation : https://documenter.getpostman.com/view/23075381/2s9YeAAa4C

## How to Run :

1. Install Node JS
2. Run `npm install`
3. Run `npm run dev`
4. Open on http://localhost:3000

## Library Use in Project :

- JSON Web Token
- Bcrypt
- Mongoose
- Body Parser
- Nodemon
